## Getting Started
First, Apply env (base on env.exmaple)
Run the development server:

```bash
npm run dev:socket
```
Run the production server:
 ```bash
npm run build:socket
npm run start:socket
```
Stop the production server:
```bash
npm run stop:socket
```
